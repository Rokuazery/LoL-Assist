# LoL Assist
LoL Assist is an automatic Runes & Spells importer for League of Legends with a simple-looking GUI.<br />
Builds are scraped from [Metasrc](http://metasrc.com). Adding more options soon!

**Features**
- Auto Import Runes
- Auto Import Spells
- Auto Accept Matchmaking
